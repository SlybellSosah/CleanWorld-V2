# Clean World Inc. - Tools Directory

This directory is designated for all standalone development scripts, automation helpers, database seed utilities, and project validation runners. 

## Purpose
Following the **BLAST (Business Logic, Architecture, Specs, Tools)** project layout, keeping executable files and scripts separated from source logic under `/tools/` preserves workspace hygiene and ensures development commands are easily discoverable.

## Permitted File Types
- Shell scripts (`*.sh`)
- Standalone CLI utilities (Node.js/Python scripts)
- Database migration seeds
- Integration test or screenshot helper scripts
